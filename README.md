## React 파일 제외
직접적인 사용 빈도가 작은 일부 파일을 VSCODE 프로젝트에서 제외시킬 수 있습니다.

```json
{
    "files.exclude": {
        "**/node_modules": true,
        "**/package.json": true,
        "**/package-lock.json": true,
        "**/.gitignore": true
    }
}
```
